This project consists of a vehicle catalog system for sale, with a public interface for viewing and an administrative panel for registering, editing and deleting records. The system uses technologies such as Node.js, React and Python (FASTAPI) to create a REST API and interact with a MySQL database and also uses JWT (JSON Web Token) for secure authentication in the administrative panel.

Technology

  • Python
  • React
  • Typescript
  • FastAPI
  • AWS S3
